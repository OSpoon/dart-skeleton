# dart-skeleton

> 此项目是在[dps](https://github.com/famanoder/dps)项目基础上重写的Ts版本，强类型的语言便于学习和入门理解。

### 构建说明：

1. 全局安装Typescript，调试代码推荐一起安装ts-node：
```sh
yarn global add typescript
```

2. 安装项目配置的依赖：
```sh
yarn
```

### 使用说明

1. 编译脚本文件：
```sh
yarn build:script
```

2. 编译源码文件：
```sh
yarn build:source
```

3. 定制配置文件dart.skeleton.config.js；

4. 启动执行：
```sh
yarn start
```
